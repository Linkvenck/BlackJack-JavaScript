# BlackJack-JavaScript
 
This game is blackjack. Made in javascript.

<b> Instructions: </b> 

Nuevo Juego - Reset the game <br>
Pedir carta - Take a card from the deck <br>
Detener     - Pass the turn to the computer <br>
